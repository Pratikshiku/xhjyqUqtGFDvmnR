## Port Ship Container Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/484c56c701614e0d880a72c9dd58ee47/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Ship Information
5. News
6. Personal Center
7. Backend Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Ship Information Management
5. Basic Data Management (Ship Type Management, Container Type Management, News Type Name Management)
6. Container Information Management
7. Route Information Management
8. News Management
9. User Management
10. Carousel Information
